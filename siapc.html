SIAP Clone

This project is a simplified clone of the SIAP Undip (Universitas Diponegoro Academic Information System) website. It was developed as a learning exercise to better understand how academic information systems work, particularly in managing student data, course registration (KRS), and academic schedules. The clone includes basic features such as a student login page, a simple dashboard displaying profile information, a form-based course registration system, and a class schedule display. While it does not replicate the full functionality of the original SIAP platform, this version focuses on core user interface elements and essential academic workflows. The project is built using HTML, CSS, and JavaScript, with optional backend integration using PHP or Node.js and MySQL. Its main purpose is to improve web development skills while simulating a real-world academic system interface.
