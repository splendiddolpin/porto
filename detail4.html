Project Overview

This project is a Hearts of Iron IV (Vanilla) game mod focused on providing various buffs and event-based cheats to help new players understand the core mechanics of the game without frustration.

Instead of changing the base gameplay drastically, this mod enhances learning by offering optional advantages such as stronger armies, faster construction, simplified diplomacy, and custom events.

Main Features
  • Army buffs (organization, attack, defense, experience)
  • Construction & infrastructure speed boost
  • Political power & stability modifiers
  • Production efficiency & factory output buffs
  • Diplomacy-related advantages for smoother gameplay
Event-Based Cheats
  • Add traits to generals and commanders
  • Instantly add resources and equipment
  • Core neighboring countries
  • Spawn divisions & stockpile equipment
  • Trigger learning-focused custom events
Tech Stack & Tools
  • Hearts of Iron IV Vanilla Modding
  • Paradox Script (.txt)
  • Custom Events & Decisions
  • Modifier & National Spirit System
Current Focus
  • Balancing buffs for learning without breaking immersion
  • Improving event clarity and descriptions
  • Expanding decision-based cheat menu
  • Ensuring full compatibility with HOI4 Vanilla
