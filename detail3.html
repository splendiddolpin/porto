Portfolio Overview

This website is my personal portfolio, designed to present my profile, skills, and projects in a clean and professional manner. It has been continuously improved to better highlight my ongoing work, especially an online book marketplace project.

Recent updates include adding a dedicated Ongoing Project section, updating the main profile photo on the homepage, and improving the overall visual consistency and user experience across all sections.

Tech Stack

  • No database website
  • Bootstrap 5
