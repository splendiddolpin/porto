Overall Progress

60% completed

Project Overview

This project is an online book marketplace that allows users to browse available books and make purchases directly through WhatsApp. Instead of a traditional checkout system, the platform simplifies transactions by redirecting buyers to WhatsApp chats with pre-filled order messages.

The website is developed using Laravel and runs on a Laragon local development environment, with a structured database to manage books, categories, and transaction records.

Tech Stack

  • Laragon (Local Development Environment)
  • MySQL Database
  • Blade Template Engine
  • Bootstrap 5
Current Focus

  • Book management (CRUD)
  • User interface and UX improvement
